== Description ==

VERSION LIGHT ONLY SHORTCODE, FULL VERSION [http://wordpress.org/extend/plugins/rebelmouse-widget/](http://wordpress.org/extend/plugins/rebelmouse-widget/)

Add your [RebelMouse](http://www.rebelmouse.com/) to your blog! RebelMouse is your social front page; connect all of your social networks to create a beautiful, dynamic site in seconds. RebelMouse shows the world what you care about and automatically updates as you share. You'll need to sign up (free) at [RebelMouse.com](http://www.rebelmouse.com/) to use the plugin. 

This plugin makes it super simple to have the front page of your WordPress site powered by RebelMouse using shortcode. 

Adding RebelMouse to your blog is quick and easy (you can find information about how to do so in the Installation section). You can email us with any questions or feedback at wordpress{at}rebelmouse.com.

== Installation ==

This plugin only support shortcode for rebelmouse full plugin go to [http://wordpress.org/extend/plugins/rebelmouse-widget/](http://wordpress.org/extend/plugins/rebelmouse-widget/)

This Wordpress plugin is easy to install:

1. Extract the contents of the zip file into your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use [shortcodes](http://support.wordpress.com/shortcodes/) to embed your rebelmouse stream in any post or page. 

Usage:

`[rebelmouse sitename="rebelmouse"]`

`[rebelmouse sitename="rebelmouse" h="1500"]`

`[rebelmouse sitename="rebelmouse" more_button=1 ]`

Arguments provides by the shortcode:

* 'site_name' You rebelmouse site name
* 'skip' Element to hide in the stream: about-site
* 'h' or 'height' height of the iframe, if is not setting had auto-growing.
* 'more_button' 1 to display a bottom button to load more posts, 0 will load automatically new posts, (default 0).
